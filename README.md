## How-to
### 1. Install NodeJS
```bash
sudo dnf install nodejs
```
### 2. Git clone this repo
```bash
git clone https://github.com/fachschaft-eit-hka/eit-kickerturnier-web.git
```
### 3. cd into the folder
```bash
cd eit-kickerturnier-web
```
### 3. install packages
```bash
npm install
```
### 4. Run dev server
```bash
npm run dev
```
